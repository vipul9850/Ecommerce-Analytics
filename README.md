#  E-commerce Pricing & Revenue Analytics

## Project Overview

This project analyzes 36 months of Indian e-commerce sales data to understand revenue growth, pricing strategies, discount impact, and regional performance.

## Tools Used

* Python (Pandas, Matplotlib)
* Power BI
* DAX
* Data Analysis

## Key Business Questions

* Which product categories generate the highest revenue?
* How do discounts impact sales volume?
* Which regions perform best in revenue generation?
* Which sales events drive the most revenue?

## Dashboard Features

* Revenue by Category
* Revenue by Zone
* Monthly Revenue Trends
* Discount vs Units Sold
* Sales Event Performance

## Key Insights

* West zone generated the highest revenue.
* Discounts above 25% significantly increased sales volume.
* Festival sales events drove major revenue spikes.
* Premium brands generated higher revenue per unit.
